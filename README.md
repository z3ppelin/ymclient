ymclient v1.0  
================================================
###A PHP client for Yahoo! Messenger API  

Features
--------------------
 - login
 - logout
 - avatar retreival
 - groups with contacts list retrieval
 - notifications retrieval (messages, buddy requests...)
 - messages sending
 - authorizing buddies
 - checking yahoo session & keeping it alive

Unit testing
--------------------
PHPUnit tests are located in `tests/` folder
